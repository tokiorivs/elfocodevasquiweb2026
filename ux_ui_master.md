# UX/UI Master Architect Rule: Antigravity + Stitch (V1.0)

## 🎯 Objetivo Primario
Transformar cada salida de diseño en una experiencia de usuario de clase mundial, priorizando la usabilidad humana, la estética emocional y el rendimiento técnico mediante la integración de Google Antigravity y Stitch.

---

## 🧠 Principios de Diseño Cognitivo
* **Ley de Jakob:** El diseño debe ser intuitivo. No reinventes la rueda a menos que la nueva sea 10x mejor.
* **Ley de Hick:** Minimiza las opciones para reducir la carga cognitiva. Menos es más.
* **Jerarquía Visual:** Utiliza el "Escaneo en F" y "Escaneo en Z" para guiar el ojo hacia las acciones clave (CTAs).
* **Regla de los 3 Clics:** Cualquier información crítica debe ser accesible en un máximo de 3 interacciones.

---

## 🎨 Estándares Estéticos y Técnicos
### 1. Sistema de Diseño (Design System)
* **Consistencia:** Obligatorio el uso de un sistema de rejilla (Grid) de 8pt.
* **Tipografía:** Escalas tipográficas basadas en ratios armónicos (ej. 1.250 / Major Third).
* **Color:** Cumplimiento estricto de accesibilidad WCAG 2.1 (Contraste mínimo 4.5:1 para texto normal).

### 2. Comportamiento en Antigravity
* **Micro-interacciones:** Cada acción debe tener un feedback visual inmediato (<100ms).
* **Continuidad:** Las transiciones entre estados deben ser fluidas, no abruptas (Easing: Cubic-Bezier).

---

## 🛠️ Protocolo de Ejecución (Stitch MCP)
Cuando el usuario solicite un diseño o interfaz, el agente debe:

1.  **Auditar Contexto:** ¿Quién es el usuario final? (Persona).
2.  **Definir User Flow:** Mapear la ruta antes de tocar el "lienzo".
3.  **Wireframing Mental:** Estructurar la información por orden de relevancia.
4.  **Generación de Alta Fidelidad:** Aplicar estilos modernos (Neumorfismo suave, Glassmorfismo o Material Design 3 según el caso).

---

## 🚫 Restricciones Críticas
* **Prohibido** usar "Lorem Ipsum". Todo el copy debe ser persuasivo y real (UX Writing).
* **Prohibido** sacrificar accesibilidad por "tendencias de diseño".
* **Prohibido** generar interfaces saturadas de elementos innecesarios.
